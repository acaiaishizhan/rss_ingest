# 📰 Feishu RSS Digest (Alibaba Cloud Edition)

![Python](https://img.shields.io/badge/Python-3.10%2B-blue?logo=python) ![Aliyun](https://img.shields.io/badge/Deployment-Alibaba_Cloud_FC-FF6600?logo=alibabacloud) ![NVIDIA](https://img.shields.io/badge/LLM-NVIDIA%20NIM-76B900?logo=nvidia) ![Feishu](https://img.shields.io/badge/Platform-Feishu%20Open-00D6B9?logo=lark)

> **让阿里云为你打工，让飞书成为你的情报司令部。**
> 这是一个基于 **阿里云函数计算 (Serverless)** 架构构建的工业级 **AI 自动化情报中台**。

---

## ⚡ 核心能力：企业级 AI 智囊团

### 🧠 深度认知：拒绝简单的搬运
本项目默认集成 **NVIDIA NIM (Qwen 80B+)** 级别大模型，依托阿里云极致的算力调度，它不是在“转发”新闻，而是在“思考”：
* **智能评分**：根据你的业务价值观自动打分，一眼洞察海量资讯中的 Top 1%。
* **多维摘要**：告别标题党，直接输出“一句话核心”与“结构化要点”。
* **语义去重**：识别洗稿与同质化内容，确保你的飞书表格里每一条都是独家情报。

### ☁️ 云端大脑：阿里云 Serverless 极致架构
* **极致弹性**：基于 **阿里云函数计算 FC 3.0**，毫秒级冷启动，按需调度，海量 RSS 源并发处理无压力。
* **极简运维**：依托 `s.yaml` 实现一键部署，无需服务器，无需配置 Python 环境，云端 7x24 小时自动运行。
* **动态思维同步**：直接在飞书文档中书写 Prompt。你改一下文档，部署在阿里云上的 AI 分析逻辑秒级同步更新。
